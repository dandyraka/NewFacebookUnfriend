Facebook Unfriend Inactive (Last Status Update)

1. Get token DISINI
2. Buka CMD/Terminal, run php unfriend.php. Input token & tahun
3. Jika proses pada CMD/Terminal selesai maka hasil akan disimpan pada inactive_id.txt
4. Upload file inactive_id.txt dibawah
5. Lalu hasilnya jalankan pada Console di Facebook
